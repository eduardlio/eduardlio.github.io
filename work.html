Random Rosetta

A random coding exercise generator.

Random Rosetta Code is a mini web-app that allows users to randomly pick a programming exercise from the Rosetta Code website so that they don't have to decide for themselves. I developed random rosetta code for myself initially but I've had friends use it for themselves.

This was a great project to work on to learn the basics of Angular 4 and to take things beyond a simple tutorial.

Convict Life

An exploratory learning tool for primary school students.

Convict Life is a university project dedicated to teaching primary school students the life of a convict during Australia's convict era. This project gave me the chance to design and develop a system that used a government API to display information in an interesting way.
